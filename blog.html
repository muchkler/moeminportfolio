Cohen Design launches on Youtube.

20 September, 2018

Youtube is definitely filled up with all kinds of tutorials nowadays; however, one of the biggest problems I see is that the tutorials are neither geared towards beginners nor more advanced programmers. Cohen Designs will solely focus on providing tutorials for beginners and more intermediate developers. The channel will cover many topics ranging from simple HTML and CSS to Node.js and Express web applications. The channel is just starting out, so there are definitely a lot of things to iron out.
